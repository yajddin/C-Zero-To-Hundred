## 060 可持久化通讯录

实现通讯录，联系人含唯一 id、动态姓名和电话。支持 ADD、UPDATE、DELETE、FIND、SAVE 和 LOAD；文件采用自定义版本化二进制格式。

#### 输入格式：

第一行输入操作数 q，随后 q 行各是一条操作；SAVE 与 LOAD 后跟文件名。

#### 输出格式：

ADD 输出 id；FIND 输出联系人或 NOT_FOUND；文件错误或格式损坏输出 IO_ERROR 或 CORRUPTED。

#### 示例 1：

> 输入：4
>
> ADD Alice 123
>
> SAVE book.bin
>
> DELETE 0
>
> LOAD book.bin
>
> 输出：0
>
> LOADED 1

#### 示例 2：

> 输入：2
>
> LOAD missing.bin
>
> FIND 9
>
> 输出：IO_ERROR
>
> NOT_FOUND

#### 提示：

- 1 ≤ q ≤ 100000。
- LOAD 必须先完整验证到临时对象，成功后再替换当前通讯录。
- 文件中使用固定宽度整数与明确字节序，不直接写入结构体原始内存。
