## 045 柔性数组数据包

定义 Packet，固定头部保存类型和负载长度，尾部使用柔性数组成员保存负载。构造数据包并计算负载字节校验和。

#### 输入格式：

第一行输入 type 和 len，第二行输入 len 个 0 到 255 的字节。

#### 输出格式：

输出 type、len 和字节和对 256 取模的结果。

#### 示例 1：

> 输入：2 4
>
> 1 2 3 255
>
> 输出：2 4 5

#### 示例 2：

> 输入：9 0
>
> 输出：9 0 0

#### 提示：

- 0 ≤ len ≤ 1000000。
- 申请大小为头部大小加负载长度，并检查加法溢出。
