## 030 位图集合

用动态字节数组实现整数集合，元素范围为 0 到 m。支持 ADD x、REMOVE x 和 HAS x。

#### 输入格式：

第一行输入 m 和操作数 q，随后 q 行各是一条操作。

#### 输出格式：

每次 HAS 输出 YES 或 NO。

#### 示例 1：

> 输入：10 4
>
> ADD 3
>
> HAS 3
>
> REMOVE 3
>
> HAS 3
>
> 输出：YES
>
> NO

#### 示例 2：

> 输入：100 2
>
> HAS 99
>
> ADD 99
>
> 输出：NO

#### 提示：

- 0 ≤ m ≤ 10^8，1 ≤ q ≤ 200000。
- 每个元素只占 1 位；注意位移使用无符号值。
