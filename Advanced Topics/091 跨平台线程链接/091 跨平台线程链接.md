## 091 跨平台线程链接

构建使用线程 API 的 worker 库和 app。通过 CMake 标准线程包表达编译、链接要求，不能硬编码 pthread 或平台库文件名。

#### 输入格式：

worker.c 创建多个线程并汇总结果，app 只链接 worker。

#### 输出格式：

支持线程的平台上配置、编译和链接成功，app 输出正确汇总值。

#### 示例 1：

> 输入：Linux 上使用 GCC 构建
>
> 输出：自动选择所需线程编译与链接参数

#### 示例 2：

> 输入：Windows 上使用 MSVC 构建
>
> 输出：不附加无效的 -pthread 参数

#### 提示：

- 使用 find_package(Threads REQUIRED) 和 Threads::Threads。
- 根据 worker 的公开 API 判断线程依赖应为 PUBLIC 还是 PRIVATE。
