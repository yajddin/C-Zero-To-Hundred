## 087 多配置生成表达式

使用生成表达式按配置控制宏与优化：Debug 定义 APP_DEBUG=1，Release 定义 NDEBUG，并让日志库只在 Debug 配置链接。

#### 输入格式：

项目可由 Ninja 单配置生成器或 Visual Studio 多配置生成器构建。

#### 输出格式：

不同配置获得正确宏和依赖，不读取或改写 CMAKE_BUILD_TYPE 来猜测多配置构建。

#### 示例 1：

> 输入：cmake --build build --config Debug
>
> 输出：程序打印 DEBUG，且链接 debuglog

#### 示例 2：

> 输入：cmake --build build --config Release
>
> 输出：程序打印 RELEASE，且不链接 debuglog

#### 提示：

- 使用 CONFIG 生成表达式。
- 生成表达式在生成阶段求值，注意参数引用与列表分隔。
