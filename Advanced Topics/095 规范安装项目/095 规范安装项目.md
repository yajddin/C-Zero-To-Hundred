## 095 规范安装项目

为 toolkit 库和 tool 可执行程序编写安装规则，分别安装运行文件、库、归档文件和公开头文件，并遵循 GNUInstallDirs。

#### 输入格式：

用户通过 CMAKE_INSTALL_PREFIX 指定任意安装前缀，可选择 Release 或 Debug 配置。

#### 输出格式：

安装树结构规范，支持 cmake --install；不把绝对本机路径写死在规则中。

#### 示例 1：

> 输入：cmake --install build --prefix stage
>
> 输出：stage 下出现 bin、lib 和 include 中的对应文件

#### 示例 2：

> 输入：cmake --install build --prefix package --config Release
>
> 输出：只安装 Release 配置产物

#### 提示：

- 使用 install(TARGETS)、install(FILES) 和 GNUInstallDirs。
- 为后续导出包正确声明 INCLUDES DESTINATION。
