## 035 引用计数字符串

设计 RcString，共享一份不可变字符数据并维护引用计数。支持 NEW、CLONE、DROP 和 PRINT，引用数降为 0 时释放内存。

#### 输入格式：

第一行输入操作数 q；每个句柄用非负整数标识，操作参数保证格式正确。

#### 输出格式：

CLONE 输出新句柄；PRINT 输出内容和当前引用数；操作无效句柄时输出 INVALID。

#### 示例 1：

> 输入：4
>
> NEW hello
>
> CLONE 0
>
> PRINT 1
>
> DROP 0
>
> 输出：0
>
> 1
>
> hello 2

#### 示例 2：

> 输入：3
>
> NEW C
>
> DROP 0
>
> PRINT 0
>
> 输出：0
>
> INVALID

#### 提示：

- 本题暂不考虑多线程。
- 句柄对象与共享控制块具有不同生命周期。
