## 089 对象库与接口库

项目有一组公共实现源码，要同时组成静态库和共享库，不能重复编译；另有纯头文件 assertions 组件负责传播宏。

使用 OBJECT 库复用对象文件，使用 INTERFACE 库表达头文件组件。

#### 输入格式：

src 目录含三个 C 文件，include 目录含公开头文件与 assertions.h。

#### 输出格式：

生成 core_static、core_shared 和 demo；公共 C 文件每种配置只编译一次。

#### 示例 1：

> 输入：构建全部目标
>
> 输出：静态库、共享库和 demo 均生成成功

#### 示例 2：

> 输入：只修改 assertions.h 后重新构建
>
> 输出：依赖该头文件的目标重新编译

#### 提示：

- OBJECT 库也需要位置无关代码才能安全用于部分共享库。
- INTERFACE 目标不产生二进制文件。
