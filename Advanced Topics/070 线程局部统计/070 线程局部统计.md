## 070 线程局部统计

把数组分给 t 个线程，使用线程局部存储记录每个线程处理的正数、负数和零的数量。线程退出前把本地结果合并。

#### 输入格式：

第一行输入 n 和 t，第二行输入 n 个整数。

#### 输出格式：

输出正数、负数和零的总数。

#### 示例 1：

> 输入：6 3
>
> -1 0 2 3 0 -4
>
> 输出：POS 2 NEG 2 ZERO 2

#### 示例 2：

> 输入：3 2
>
> 1 2 3
>
> 输出：POS 3 NEG 0 ZERO 0

#### 提示：

- 1 ≤ n ≤ 10000000，1 ≤ t ≤ 64。
- 热循环中不访问共享计数器，合并阶段才同步。
