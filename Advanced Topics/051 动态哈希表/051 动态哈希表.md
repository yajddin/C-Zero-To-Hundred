## 051 动态哈希表

用拉链法实现字符串到整数的哈希表，支持 SET key value、GET key 和 DELETE key。负载因子超过 0.75 时扩容并重新散列。

#### 输入格式：

第一行输入操作数 q，随后 q 行各是一条操作，key 不含空格。

#### 输出格式：

GET 输出值或 NOT_FOUND；DELETE 不存在的键时输出 NOT_FOUND。

#### 示例 1：

> 输入：4
>
> SET age 18
>
> GET age
>
> SET age 20
>
> GET age
>
> 输出：18
>
> 20

#### 示例 2：

> 输入：2
>
> GET none
>
> DELETE none
>
> 输出：NOT_FOUND
>
> NOT_FOUND

#### 提示：

- 1 ≤ q ≤ 200000，key 长度不超过 63。
- 表必须拥有键的副本；扩容失败时保留原表可用。
